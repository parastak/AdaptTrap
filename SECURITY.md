# Security Policy

## Supported Versions

AdaptTrap is an open-source research project.
Security fixes will generally be applied to the latest version available on the main branch.

---

## Reporting a Vulnerability

If you discover a security issue within AdaptTrap itself, please avoid creating a public issue.

Instead:

- Open a private security advisory through GitHub (if enabled), or
- Contact the maintainer directly.

Please include:

- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

We aim to acknowledge reports within a reasonable timeframe.

---

## Responsible Use

AdaptTrap is intended for:

- Research
- Education
- Defensive Security
- Reinforcement Learning Research
- Cyber Deception Research

It is **not** intended to be used against systems you do not own or have explicit authorization to test.
Users are responsible for complying with applicable laws and regulations.

---

## Scope

This repository contains research code and simulation environments.
It should not be considered production-ready security software.
No warranty is provided regarding operational security or suitability for deployment in production environments.
